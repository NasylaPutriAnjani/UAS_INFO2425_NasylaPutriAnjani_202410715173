# Panduan Pengguna (User Manual) - RubbyBooks

## Cara Registrasi & Login

1.  Klik menu **Masuk / Daftar** di sudut kanan atas pada tampilan desktop (atau melalui navigasi pada tampilan mobile).
2.  Sebuah popup (*modal*) akan muncul. Di sana Anda dapat memilih formulir **Masuk** jika sudah punya akun, atau **Daftar** untuk membuat akun baru.
3.  Untuk Pembeli, biarkan tab role pada **Pembeli**. Untuk mendaftar sebagai toko buku, pilih tab **Penjual**.
4.  *(Khusus Penjual)* Akun yang baru saja didaftarkan sebagai penjual akan berada pada status *Pending*, dan menunggu konfirmasi dari Admin sebelum bisa berjualan.

## Panduan untuk Pembeli (Buyer)

*   **Pencarian Buku:** Gunakan kolom pencarian di bagian navigasi atas, atau klik tombol "Katalog" untuk melihat daftar buku berdasarkan kategori, harga, maupun rating ulasan.
*   **Wishlist:** Menemukan buku yang disukai namun belum ingin dibeli? Klik logo "♡" (hati) pada sampul buku. Buku akan otomatis tersimpan di halaman Wishlist yang bisa diakses di panel Akun.
*   **Keranjang & Checkout:** Klik tombol "+ Keranjang" atau icon troli untuk menyimpan buku. Buka jendela keranjang (icon troli di kanan atas) lalu klik "Lanjut ke Checkout". Lengkapi form alamat penerima (ongkos kirim akan dihitung otomatis sesuai kota) lalu lakukan konfirmasi pesanan.
*   **Pembayaran:** Setelah mendapatkan nomor pesanan (*Invoice*), buka menu **Pesanan Saya**, lalu unggah gambar bukti struk pembayaran pada pesanan dengan status "Pending".

## Panduan untuk Penjual (Seller)

*   **Menambah Produk:** Masuk ke *Dashboard Penjual*, pilih menu "Produk Saya" lalu tekan tombol "+ Tambah Produk". Isi informasi detail buku, kategori, ketersediaan stok, harga, dan sampul. Jika sampul tidak diisi, akan otomatis menggunakan *placeholder*.
*   **Memproses Pesanan:** Buka menu "Pesanan Masuk". Anda akan melihat seluruh rincian transaksi pembeli. Jika pembayaran pembeli sudah valid, klik Edit lalu ubah status pesanan menjadi "Diproses". Ketika buku sudah dikirim via kurir, ubah status menjadi "Dikirim" beserta input nomor resi pelacakan.
*   **Laporan Keuangan:** Anda dapat memantau pergerakan omzet mingguan, bulanan, maupun tahunan dengan masuk ke menu "Laporan Penjualan". Grafik visual akan memudahkan evaluasi penjualan Anda.

## Panduan untuk Admin

*   **Memonitor Aktivitas:** Pada *Dashboard Admin*, Anda bisa memantau statistik total penjualan secara langsung *(real-time)*.
*   **Verifikasi Seller Baru:** Di halaman "Pengguna", cari baris pengguna berstatus "Penjual (Pending)" kemudian klik aksi setuju (Approve) untuk memberikan hak operasi toko.
*   **Keamanan Akun:** Admin berhak melakukan proses pemblokiran akun *(banned)* terhadap pembeli/penjual yang melanggar ketentuan.
